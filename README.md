# CSSummerCampDDL

本仓库为夏令营截止日期DDL静态网页。

暂时为基础网页版本，支持信息的格式化渲染以及动态排序，今后考虑加入如开营信息/预推免Version等内容。

本网站全部信息来自CSBAOYAN的[CSSummerCamp2024](https://github.com/CS-BAOYAN/CSSummerCamp2024)。
